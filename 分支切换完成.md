## 🔧 下一步：在 GitHub 上设置默认分支

虽然远程只有 main 分支，但建议在 GitHub 上明确设置默认分支：

### 步骤：

1. **访问仓库设置**
   ```
   https://github.com/ffzzffz/studyroom/settings
   ```

2. **找到 "Branches" 选项**
   - 点击左侧菜单的 `Branches`

3. **设置默认分支**
   - 在 "Default branch" 部分
   - 如果显示的不是 `main`，点击切换按钮
   - 选择 `main` 作为默认分支
   - 点击 `Update` 确认

- ✅ **GitHub 默认分支** - 2020 年后 GitHub 新仓库默认使用 `main`

- ✅ **更具包容性** - 符合技术社区的多元化倡议
